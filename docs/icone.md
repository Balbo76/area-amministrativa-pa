1. ICONE AREE DI EMERGENZA (Cartografia e Piani Comunali)
------------------------------------------------------------------------
- 'area-attesa'      -> Area di Attesa Popolazione (Segnale "A")
Uso: Punti di raccolta sicuri nei quartieri. 
- 'area-ricovero'     -> Area di Ricovero e Accoglienza (Tenda)
Uso: Tendopoli, palestre, hub per sfollati.
- 'area-ammassamento' -> Area Ammassamento Soccorritori (Camion/Hub)
Uso: Logistica, colonne mobili, mezzi di soccorso.

2. CENTRI DI COMANDO E COORDINAMENTO

- 'coc'              -> Centro Operativo Comunale (Sede con Antenna) 
- 'com'              -> Centro Operativo Misto (Sede con Antenna)
Uso: Sale operative di gestione emergenza.
- 'sede-volontari'    -> Sede del Gruppo / Associazione (Casa con Scudo)
Uso: Presidi fisici delle tute gialle/blu.

---
3. RISCHI AMBIENTALI (Per Bollettini CFD Veneto e Widget Meteo)

- 'rischio-idrogeologico' -> Alluvione / Allertamento Idrico (Onde)
- 'rischio-meteo'         -> Temporali / Vento Forte / Vaia (Nuvola + Fulmine)
- 'rischio-sismico'       -> Terremoto / Scosse (Terreno con Faglia)
- 'rischio-incendi'       -> Incendi Boschivi (Fiamma / Albero)
- 'rischio-neve'          -> Neve / Ghiaccio / Valanghe (Cristallo/Fiocco)

---
4. UTILITY WEB & CALL-TO-ACTION (Interfaccia Portale AgID)

- 'allerta-attiva'     -> Stato di Allerta in corso (Campanella/Sirena)
Uso: Header banner e notifiche push urgenti.
- 'norme-comportamento' -> Campagna "Io Non Rischio" (Libro + Info)
Uso: Didattica su cosa fare prima/durante/dopo.
- 'diventa-volontario'  -> Iscrizione Nuovi Volontari (Utente + Plus)
Uso: Call-to-action per l'arruolamento nel sistema.
